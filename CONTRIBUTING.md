# Contributing to VS Code Copilot Customizations

Thank you for your interest in contributing! This framework helps developers work effectively with long-running AI coding agents.

## How to Contribute

### Reporting Issues

- Use GitHub Issues for bugs or feature requests
- Include VS Code version and Copilot extension version
- Describe expected vs actual behavior

### Submitting Changes

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Test that prompts work in VS Code Agent mode
5. Commit with conventional commit messages (`feat:`, `fix:`, `docs:`)
6. Push and open a Pull Request

### Areas for Contribution

- **New prompt files** for common workflows
- **Instruction files** for specific languages/frameworks
- **Template improvements** for different project types
- **Documentation** clarifications and examples
- **Bug fixes** in existing prompts

### Guidelines

- Follow existing file naming conventions
- Include frontmatter in prompt/instruction files
- Test prompts in actual Agent mode sessions
- Update README if adding new features

## Code of Conduct

Be respectful and constructive. We're all here to make AI-assisted development better.
